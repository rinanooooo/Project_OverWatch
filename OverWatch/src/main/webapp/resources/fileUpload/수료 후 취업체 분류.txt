[수료 후 취업체 분류]

1. 솔루션 업체(Solution) : 대행업체(종류 무관)도 있으나 극소수이고
		        자체 프로그램을 갖고 있음.(통계, ERP, 자원관리, ...)

2. 에이전시(Agency) : 대행업체 (홈페이지, 쇼핑몰, 모바일 앱)

3. SI : 파견업체(=용역업체), 개발업무

4. SM : 파견업체(=용역업체), 개발완료 후 유지/보수
	(참고 - 대부분의 업체가 SI/SM 를 함께 적용하고 있음)

5. 일반업체 전산직 : 쇼핑몰 관리자 등을 말함.

6. 전산직 계열 공공기관 : 계약직(자격증, 졸업증명서, ...)
		=> 해당 공공기관 웹사이트에서 비정기적으로 공고를 띄움.

7. 호스팅 업체 : 웹사이트를 만드는 컴퓨터를 제공하는 업체
   공인 IP, 호스팅 서비스가 제공되는 인터넷 서비스
